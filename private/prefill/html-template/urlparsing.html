Extracting data from Urls

This example shows you how to extract data from the browser's url. This can be useful in many cases. For example, when you create multiple QR codes, with different parameters, but having all of them pointing to the same base URL.

This page will show you any URL parameter. Including full url, protocol, port, path, hash, query parameters, etc.
